# Routing Rules

## Core Rule

Route each request to the most appropriate agent or workflow based on the task type.

## Suggested Routing

- Documentation and planning → Coordinator
- Research and investigation → Researcher
- Code changes and implementation → Builder
- Quality review and validation → Reviewer

## Process

1. Identify the primary objective.
2. Classify the request by type.
3. Assign the most relevant role.
4. Confirm handoff details if the task continues across agents.
